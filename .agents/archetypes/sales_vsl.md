# Sales / VSL

Use when the footage is meant to persuade, sell, handle objections, or move the viewer toward an offer.

Common structure:
HOOK -> PAIN -> AGITATION -> SOLUTION -> PROOF -> OFFER / CTA

Prioritize:
- direct pain statement;
- high-clarity promise;
- proof or credibility;
- concise explanation of the mechanism;
- clean CTA.

Cut:
- vague claims;
- repeated setup;
- tangents that delay the offer;
- weak proof moments;
- over-explaining after the viewer already understands.

Common mistakes:
- keeping too much context before the pain;
- losing proof/result moments;
- making the CTA abrupt or unsupported.

Target runtime guidance:
- fit the brief if given; otherwise preserve enough persuasion logic to make the offer coherent.

Do not force all beats if the material does not support them.
