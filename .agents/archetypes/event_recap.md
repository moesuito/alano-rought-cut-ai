# Event Recap / Narrative Recap

Use when footage summarizes an event, trip, workshop, launch, or shared experience.

Common structure:
ARRIVAL / SETUP -> HIGHLIGHTS -> HUMAN MOMENTS -> RESULT / CLOSING

Prioritize:
- setup that establishes where/why;
- highlight moments;
- human reactions;
- result, takeaway, or closing reflection;
- natural energy shifts.

Cut:
- repetitive arrival/setup;
- logistical details without story value;
- dead time between highlights;
- weak duplicate reactions.

Common mistakes:
- keeping chronological order when story order is clearer;
- cutting all quiet moments;
- ending without a result or emotional close.

Target runtime guidance:
- depends on use; keep enough shape to feel like a recap, not random clips.

Do not force all beats if the material does not support them.
