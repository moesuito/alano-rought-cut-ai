# Tutorial

Use when the footage teaches a process, setup, workflow, or sequence of actions.

Common structure:
INTRO -> SETUP -> STEPS -> GOTCHAS -> RESULT -> RECAP

Prioritize:
- clear setup requirements;
- ordered steps;
- warnings or gotchas that prevent failure;
- result demonstration;
- concise recap.

Cut:
- repeated explanations of the same step;
- dead time between actions;
- mistakes unless useful as warnings;
- tangents unrelated to completing the task.

Common mistakes:
- cutting too aggressively and breaking comprehension;
- removing setup needed for later steps;
- preserving every pause during screen/process work.

Target runtime guidance:
- preserve enough length for clarity rather than forcing short-form pacing.

Do not force all beats if the material does not support them.
