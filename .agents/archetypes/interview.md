# Interview / Q&A

Use when one or more speakers answer questions or discuss topics through prompts and followups.

Common structure:
QUESTION -> ANSWER -> FOLLOWUP, repeated by topic

Prioritize:
- clean question context when needed;
- complete answers;
- strong followups;
- topic grouping;
- moments where the speaker is specific, direct, or emotionally clear.

Cut:
- repeated questions;
- rambling lead-ins;
- interviewer acknowledgements that do not add meaning;
- duplicate answers;
- tangents outside the chosen topic.

Common mistakes:
- cutting the question when the answer needs it;
- preserving recording order instead of topic order;
- dropping reaction beats that make the answer human.

Target runtime guidance:
- depends on topic depth; keep each topic coherent.

Do not force all beats if the material does not support them.
